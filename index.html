Data Analytics Showcase

I don't just find data; I question it. Here is an example of a sales analysis query and how I visualize the results in Power BI.

SQL Query: Sales Performance 
SELECT 
    Product_Category,
    SUM(Revenue) AS Total_Revenue,
    COUNT(DISTINCT Customer_ID) AS Unique_Buyers
FROM 
    Sales_Data_2024
WHERE 
    Region = 'North America'
GROUP BY 
    Product_Category
ORDER BY 
    Total_Revenue DESC;

Analysis Insight

The query reveals that while Electronics drive the most revenue, Home Goods have a higher repeat purchase rate. I recommended bundling these categories to increase average order value.
